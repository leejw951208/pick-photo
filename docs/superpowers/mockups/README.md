# Pick Photo UI 목업

이 폴더는 제품 설계와 구현 계획을 결정하기 전에 비교한 시각 목업을 보관한다. 목업은 제품 요구사항의 원천이 아니며, 최종 범위와 검증 기준은 `docs/superpowers/specs/`와 `docs/superpowers/plans/` 문서가 기준이다.

## 현재 목업

- `2026-05-01-pick-photo-ui-redesign-options.html`: `Studio Glass`, `Warm Editorial`, `Fresh Clarity` 세 방향을 비교한 HTML 목업.
- `2026-05-01-pick-photo-ui-redesign-options.png`: 위 HTML 목업의 스크린샷.

## 사용 기준

- 목업은 시각 방향 확인용이다.
- 개인정보 보관, 삭제, 공식 증명사진 규격, API 동작 같은 제품/기술 결정을 목업 문구에서 확정하지 않는다.
- 실제 구현 기준은 선택된 설계 문서와 구현 계획에 기록한다.
